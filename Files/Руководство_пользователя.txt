============================================================
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Программа «Домоуправление»
============================================================

1. ОБЩЕЕ НАЗНАЧЕНИЕ ПРОГРАММЫ
------------------------------------------------------------
Программа «Домоуправление» предназначена для ведения учета
квартир и жильцов многоквартирного дома. Программа позволяет
просматривать, добавлять, изменять, удалять записи, выполнять
поиск, фильтрацию, экспорт данных и просмотр статистики.

Все данные хранятся в CSV-файле
CFrankKTyuiu.FrankK.Sprint7FilesHousingManagement.csv


2. ГЛАВНОЕ ОКНО ПРОГРАММЫ
------------------------------------------------------------
Главное окно содержит

- меню управления
- панель поиска по фамилии
- таблицу с данными квартир

В таблице отображаются следующие поля
- Подъезд
- Квартира
- Общая площадь
- Жилая площадь
- Количество комнат
- Фамилия жильца
- Дата регистрации
- Количество членов семьи
- Количество детей
- Наличие задолженности
- Примечание


3. МЕНЮ «ГЛАВНАЯ»
------------------------------------------------------------
В меню «Главная» доступны следующие пункты

● Добавить  
Открывает окно добавления новой записи.

● Изменить  
Открывает окно редактирования выбранной записи в таблице.

● Удалить  
Удаляет выбранную запись после подтверждения.

● График  
Открывает окно с гистограммой распределения квартир
по количеству комнат.

● Экспорт CSV  
Позволяет сохранить текущие данные таблицы в CSV-файл.
Открывается диалог сохранения файла.
Имя файла по умолчанию
таблица_домоуправление_ДД-ММ-ГГГГ.csv

● Статистика  
Открывает диалоговое окно со статистикой по текущей таблице
- общее количество записей
- максимальная, минимальная и средняя общая площадь
- максимальная, минимальная и средняя жилая площадь
- фамилия жильца из самой многодетной семьи

Также доступна кнопка сохранения статистики в TXT-файл.
Имя файла по умолчанию
статистика_ДД-ММ-ГГГГ.txt

● Фильтр  
Открывает модальное окно фильтрации данных.

● О программе  
Показывает информацию о программе.

● Выход  
Закрывает программу.


4. ДОБАВЛЕНИЕ И РЕДАКТИРОВАНИЕ ЗАПИСЕЙ
------------------------------------------------------------
При добавлении или редактировании записи открывается отдельное
модальное окно.

В окне доступны следующие поля
- Подъезд
- Квартира
- Общая площадь
- Жилая площадь
- Количество комнат
- Фамилия жильца
- Дата регистрации
- Количество членов семьи
- Количество детей
- Чекбокс «Есть задолженность»
- Примечание

Кнопки
- Сохранить — сохраняет изменения
- Отмена — закрывает окно без сохранения

Окно открывается по центру экрана и не может быть растянуто.


5. ПОИСК ПО ФАМИЛИИ
------------------------------------------------------------
В верхней части главного окна находится панель поиска.

● Введите текст в поле «Поиск по фамилии»
Таблица автоматически отфильтруется по введенному тексту.

● Кнопка «Сбросить»
Очищает поле поиска и возвращает отображение всех записей.


6. ФИЛЬТРАЦИЯ ДАННЫХ
------------------------------------------------------------
Кнопка «Фильтр» открывает модальное окно фильтрации.

Доступные фильтры
- Подъезд (диапазон от и до)
- Квартира (диапазон от и до)
- Общая площадь (диапазон от и до)
- Жилая площадь (диапазон от и до)
- Количество комнат (диапазон от и до)
- Фамилия (по тексту)
- Дата регистрации (период от и до)
- Количество членов семьи (диапазон)
- Количество детей (диапазон)
- Задолженность (Да  Нет  оба)
- Примечание (по тексту)

Кнопки
- Применить — применяет фильтр к таблице
- Сбросить — сбрасывает фильтр и показывает все записи

При повторном открытии окна фильтра все поля очищены.


7. ГРАФИК
------------------------------------------------------------
При выборе пункта «График» открывается окно с гистограммой.

На гистограмме отображается
- ось X — количество комнат
- ось Y — количество квартир

Гистограмма строится автоматически по текущим данным таблицы.


8. ЭКСПОРТ В CSV
------------------------------------------------------------
Пункт «Экспорт CSV» сохраняет текущие данные таблицы в файл.

Особенности
- экспортируются только текущие данные (с учетом поиска и фильтра)
- данные сохраняются в формате CSV
- разделитель — запятая
- каждая запись сохраняется в отдельной строке


9. СТАТИСТИКА
------------------------------------------------------------
Окно статистики отображает
- общее количество записей
- максимальную, минимальную и среднюю общую площадь
- максимальную, минимальную и среднюю жилую площадь
- фамилию жильца с наибольшим количеством членов семьи

Статистику можно сохранить в TXT-файл.


10. ХРАНЕНИЕ ДАННЫХ
------------------------------------------------------------
Все изменения (добавление, редактирование, удаление) сразу
сохраняются в CSV-файл

CFrankKTyuiu.FrankK.Sprint7FilesHousingManagement.csv


11. ЗАВЕРШЕНИЕ РАБОТЫ
------------------------------------------------------------
Для завершения работы используйте пункт меню «Выход»
или стандартную кнопку закрытия окна.
